## גישה לדוח המחקר

### שלב 1: קבלת הקישור והסיסמה

תקבלו מאהיה (חוקר הסטטיסטיקה) שני דברים:

1. **קישור ייחודי** - לדוגמה:
   ```
   https://statviz.xyz/preview/ABC123XYZ
   ```

2. **סיסמה** - סיסמה אישית לפרויקט שלכם

**חשוב:** שמרו את הסיסמה במקום בטוח! תצטרכו אותה בכל פעם שתרצו לצפות בדוח.

### שלב 2: פתיחת הקישור

1. לחצו על הקישור שקיבלתם (או העתיקו אותו לדפדפן)
2. הקישור יעבוד במחשב, טאבלט, או טלפון נייד
3. מומלץ להשתמש בדפדפן מעודכן:
   - **iPhone/iPad:** Safari
   - **Android:** Chrome
   - **מחשב:** Chrome, Firefox, Edge, או Safari

### שלב 3: הזנת סיסמה

![מסך הזנת סיסמה](password-prompt-example.png)

1. תופיע מסך עם שדה סיסמה
2. הקלידו את הסיסמה שקיבלתם מאהיה
3. **טיפ:** לחצו על האייקון של העין כדי לראות את הסיסמה בזמן ההקלדה
4. לחצו על כפתור "כניסה"

**אם הסיסמה נכונה:**
- הדוח האינטראקטיבי ייפתח אוטומטית
- תוכלו לצפות בו במשך 24 שעות ללא צורך להזין סיסמה שוב

**אם הסיסמה שגויה:**
- תופיע הודעה: "סיסמה שגויה"
- נסו שוב (ודאו שאין רווחים או תווים מיותרים)

## צפייה בדוח האינטראקטיבי

### מבנה הדוח

לאחר הכניסה, תראו:

![מסך הדוח](viewer-example.png)

**1. כותרת הפרויקט** (למעלה)
- שם הפרויקט
- שם הסטודנט (שמכם)
- נושא המחקר

**2. הדוח האינטראקטיבי** (במרכז)
- דוח HTML עם גרפים אינטראקטיביים
- תוצאות ניתוח סטטיסטי
- טבלאות נתונים

**3. כפתור הורדה** (למטה במובייל / למעלה בצד במחשב)
- לחצו כדי להוריד את מסמך הממצאים (DOCX)

### ניווט בדוח

**במחשב:**
- גללו למעלה ולמטה באמצעות העכבר או גלגל העכבר
- הדוח מוצג בחלון מוטמע

**בטלפון נייד:**
- גללו באצבע למעלה ולמטה
- ניתן להגדיל/להקטין על ידי צביטה (pinch)

## שימוש בגרפים האינטראקטיביים

הדוח שלכם כולל גרפים אינטראקטיביים שנוצרו באמצעות ספריית Plotly. הנה מה שאפשר לעשות איתם:

### במחשב (עכבר)

**1. הצגת פרטים על נקודת נתונים:**
- העבירו את העכבר מעל נקודה בגרף
- יופיע תיבת מידע עם הערכים המדויקים

**2. הגדלה (Zoom):**
- גללו בגלגל העכבר מעל הגרף
- לחצו וגררו על אזור מסוים בגרף כדי להגדיל אותו

**3. הזזה (Pan):**
- לחצו וגררו את הגרף כדי להזיז אותו

**4. איפוס התצוגה:**
- לחצו פעמיים על הגרף כדי לחזור לתצוגה המקורית

### בטלפון נייד (מגע)

**1. הצגת פרטים על נקודת נתונים:**
- געו (tap) בנקודה בגרף
- יופיע תיבת מידע עם הערכים

**2. הגדלה (Zoom):**
- צבטו (pinch) את המסך באצבעות כדי להגדיל או להקטין

**3. הזזה (Pan):**
- גררו באצבע אחת כדי להזיז את הגרף

**4. איפוס התצוגה:**
- געו פעמיים (double-tap) על הגרף

### תכונות נוספות בגרפים

**כפתורים בפינה העליונה של כל גרף:**
- 📷 **צילום מסך:** שמור את הגרף כתמונה
- 🔍 **זום:** כלי להגדלה ידנית
- 📊 **בחירת נתונים:** בחרו/הסתירו קטגוריות מסוימות
- 🏠 **בית:** חזרה לתצוגה מקורית

## הורדת מסמך הממצאים

### שלב 1: איתור כפתור ההורדה

**בטלפון נייד:**
- הכפתור נמצא בתחתית המסך
- טקסט: "הורד מסמך מלא"
- אייקון של חץ למטה

**במחשב:**
- הכפתור נמצא בפינה הימנית העליונה
- טקסט: "הורד מסמך מלא"

### שלב 2: לחיצה על הכפתור

1. לחצו על כפתור "הורד מסמך מלא"
2. יופיע הודעה: "הקובץ הורד בהצלחה"
3. הקובץ יישמר במכשיר שלכם

### שלב 3: פתיחת הקובץ

**סוג הקובץ:** מסמך Word (DOCX)

**איפה הקובץ נמצא?**

**iPhone/iPad:**
- אפליקציית "Files" → תיקיית "Downloads"
- או: Safari → אייקון ההורדות (למטה)

**Android:**
- אפליקציית "Files" או "Downloads"
- או: Chrome → תפריט → Downloads

**מחשב:**
- תיקיית Downloads (הורדות)
- או: דפדפן → היסטוריית הורדות (Ctrl+J)

**שם הקובץ:**
```
[שם-הפרויקט]_findings.docx
```

### שלב 4: קריאת הקובץ

**פתיחת הקובץ:**
- Microsoft Word (מומלץ)
- Google Docs
- LibreOffice Writer
- כל תוכנת עריכת מסמכים שתומכת ב-DOCX

**הקובץ מכיל:**
- סיכום הממצאים העיקריים
- טבלאות תוצאות
- פרשנות סטטיסטית
- המלצות (במידת הצורך)

## פתרון בעיות נפוצות

### בעיה: "סיסמה שגויה"

**פתרון:**
1. ודאו שהעתקתם את הסיסמה בצורה מדויקת (ללא רווחים)
2. שימו לב להבדל בין אותיות גדולות וקטנות (A ≠ a)
3. בדקו שלא העתקתם תווים מיותרים (נקודה בסוף, וכו')
4. נסו להקליד את הסיסמה ידנית במקום להעתיק

**אם עדיין לא עובד:**
- פנו לאהיה לקבלת סיסמה חדשה

### בעיה: "הפגישה פגה תוקף"

**מה זה אומר?**
עברו 24 שעות מאז הכניסה האחרונה שלכם.

**פתרון:**
1. הזינו את הסיסמה שוב
2. תקבלו גישה ל-24 שעות נוספות

### בעיה: "שגיאת רשת"

**פתרון:**
1. בדקו את החיבור לאינטרנט
2. נסו לרענן את הדף (F5 או לחצן רענון)
3. נסו לסגור ולפתוח מחדש את הדפדפן
4. אם הבעיה נמשכת, חכו מספר דקות ונסו שוב

### בעיה: הדוח לא נטען / מסך ריק

**פתרון:**
1. חכו 10-15 שניות (קבצים גדולים לוקחים זמן לטעינה)
2. בדקו את החיבור לאינטרנט
3. נסו לרענן את הדף
4. אם המסך נשאר ריק לאחר 20 שניות, יופיע כפתור "פתח בחלון חדש"
5. לחצו על הכפתור כדי לפתוח את הדוח בכרטיסייה חדשה

### בעיה: הגרפים לא אינטראקטיביים

**סיבות אפשריות:**
- דפדפן ישן מדי
- JavaScript מושבת

**פתרון:**
1. עדכנו את הדפדפן לגרסה עדכנית
2. ודאו ש-JavaScript מופעל בהגדרות הדפדפן
3. נסו דפדפן אחר (Chrome, Safari, Firefox)

### בעיה: ההורדה לא עובדת

**פתרון:**
1. ודאו שיש לכם חיבור אינטרנט יציב
2. בדקו שלא חלפו 24 שעות מהכניסה (אם כן, התחברו שוב)
3. נסו שוב - לפעמים יש עיכוב קצר
4. בדקו את הגדרות הדפדפן (חסימת הורדות אוטומטית)

### בעיה: הקובץ הורד אבל לא נפתח

**פתרון:**
1. ודאו שיש לכם תוכנה לפתיחת קבצי Word (Word, Google Docs, וכו')
2. נסו להעלות את הקובץ ל-Google Drive ולפתוח ב-Google Docs
3. בדקו שהקובץ הורד במלואו (גודל הקובץ לא 0KB)

### בעיה: הטקסט בעברית מופיע הפוך

**פתרון:**
- זו כנראה בעיה בתוכנת הקריאה שלכם
- נסו לפתוח ב-Microsoft Word או Google Docs
- בWord: Home → Paragraph → Right-to-Left

## טיפים שימושיים

### 1. שמירת הקישור

**מומלץ:**
- שמרו את הקישור ב-Bookmarks (סימניות) של הדפדפן
- או שלחו לעצמכם במייל
- זה יאפשר לכם גישה קלה בעתיד

### 2. צפייה במכשירים שונים

**ניתן לצפות בדוח:**
- במחשב (לתצוגה מלאה ונוחה)
- בטאבלט (תצוגה נוחה לקריאה)
- בטלפון (גישה בכל מקום)

**הסיסמה עובדת בכל המכשירים!**

### 3. שיתוף עם מנחה/ים

**אם אתם רוצים לשתף את הדוח:**
1. תוכלו לשלוח את הקישור והסיסמה למנחה שלכם
2. או להוריד את קובץ ה-DOCX ולשלוח אותו במייל

**שימו לב:** כל מי שיודע את הקישור והסיסמה יכול לצפות בדוח.

### 4. גיבוי

**מומלץ:**
- הורידו את קובץ ה-DOCX ושמרו אותו במחשב/ענן
- זה יבטיח שיהיה לכם גישה לממצאים גם בעתיד

## שאלות נפוצות

### ש: כמה זמן הגישה לדוח תקפה?

**ת:** 24 שעות מרגע ההזדהות. לאחר מכן תצטרכו להזין את הסיסמה שוב.

### ש: האם אני יכול לשתף את הסיסמה?

**ת:** כן, אבל בזהירות. כל מי שמקבל את הקישור והסיסמה יוכל לצפות בדוח. מומלץ לשתף רק עם המנחה או בני משפחה קרובים.

### ש: מה עושים אם שכחתי את הסיסמה?

**ת:** פנו לאהיה. הוא יכול לספק לכם את הסיסמה שוב או ליצור סיסמה חדשה.

### ש: האם יש מגבלה על מספר הצפיות?

**ת:** לא. תוכלו לצפות בדוח כמה פעמים שתרצו, כל עוד הגישה פעילה.

### ש: האם אפשר להדפיס את הדוח?

**ת:** כן! הורידו את קובץ ה-DOCX ופתחו אותו ב-Word. משם תוכלו להדפיס (File → Print).

### ש: האם הדוח שומר את השינויים שאני עושה בגרפים?

**ת:** לא. השינויים (הגדלה, הזזה) הם זמניים ונועדו לצפייה בלבד. לכל מי שנכנס לדוח יראה את התצוגה המקורית.

### ש: האם יש גרסה לעיוורים?

**ת:** הדוח האינטראקטיבי מותאם למסכים. קובץ ה-DOCX ניתן לקריאה על ידי תוכנות קריאת מסך (Screen Readers).

### ש: האם הפלטפורמה תומכת בעברית?

**ת:** כן! כל הממשק בעברית, והדוח יכול להכיל טקסט בעברית.
